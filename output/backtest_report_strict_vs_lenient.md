# FCF策略回测报告：严格5yr OCF vs 宽松5yr OCF

> 统一起始日期：2016-06-13 | 对比区间：2016-06-13 至 2026-03-16（约10年）

## 1. 规则变更说明

| 项目 | 严格模式 | 宽松模式 |
|------|---------|---------|
| 5yr OCF检查 | 要求 base_year-4 到 base_year 共5年**均有数据且OCF>0** | 允许跳过缺失年份（上市不足5年不再一刀切排除） |
| 缺失年份处理 | `ocf is None → return False`（不通过） | `ocf is None → continue`（跳过） |
| ocf_start截断 | **不截断**，始终从 base_year-4 开始检查 | 截断到上市年：`max(ocf_start, list_year)` |
| 效果 | 上市不足5年的标的被排除 | 上市不足5年的标的仅用已有年份检查 |

## 7. 结论与建议

### ZZ800 FCF：严格5yr OCF表现较差

- 严格年化 10.48% vs 宽松年化 14.02%，宽松高出 **3.55个百分点**
- 宽松模式允许上市不足5年的优质标的入选，增加了样本多样性
- 2018-2019期间严格版本曾短暂领先，但2024-2026期间宽松大幅拉开差距

### HS300 FCF：严格5yr OCF表现更差

- 严格年化 10.84% vs 宽松年化 14.69%，宽松高出 **3.86个百分点**
- 严格规则导致HS300多个期间持仓不足50只（2018-12仅32只、2019-03仅28只）
- 持仓不足降低了分散化效果，拖累整体收益

### 建议

1. **两个策略均应采用宽松5yr OCF规则**
2. 对于HS300，建议额外增加持仓数保护机制：当严格规则下候选不足40只时，回退到宽松规则补充候选
3. 宽松规则的核心优势：允许中国海油、中国移动等高质量次新标的入选，这些标的往往有极强的FCF能力
