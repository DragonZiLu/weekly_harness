## 二、盈利质量(PQ)分析 — ⚠️ 关键问题

### PQ公式

```
PQ = (OCF - oper_profit) / total_assets
```

- `OCF`：经营现金流（TTM）
- `oper_profit`：营业利润（收入-成本-税金-销售费-管理费-财务费+投资收益）
- `total_assets`：总资产

### PQ值分布（B版，2026-03-16）

| 指标 | 值 |
|------|-----|
| PQ均值 | 0.065 |
| PQ中位数 | 0.069 |
| PQ最小值 | -0.017 |
| PQ<0.1占比 | 82% |
| PQ<0占比 | 10% |

> **解读**：PQ值普遍在0.01-0.10范围是**公式设计的正常值**，不是"差"。
> 例如格力：OCF≈600亿，oper_profit≈350亿，差额≈250亿，total_assets≈4000亿 → PQ≈0.06
> PQ≈0意味着利润与现金流基本匹配；PQ>0意味着现金流超过利润（好）；PQ<0意味着利润超过现金流（需关注）

### PQ筛选阈值

| 参数 | 值 | 说明 |
|------|-----|------|
| pq_cutoff_percentile | 20% | 淘汰PQ最低的20% |
| P20阈值(含金融地产) | **-0.0016** | 全样本计算（当前设计） |
| P20阈值(仅非金融) | 更高 | 不含金融时更严格 |

> **设计决策**（实验结论）：PQ基数含金融地产反而**提升回测表现**（CAGR 15.3%→14.2%减，夏普0.81→0.76减），因金融地产PQ普遍极低，拉低阈值使更多非金融股通过。
