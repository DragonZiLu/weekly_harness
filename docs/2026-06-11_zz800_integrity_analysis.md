## 一、方法论

### 现金实现率

```
现金实现率 = 最近 2 个会计年度累计 FCF / 累计归母净利润
          = Σ(OCF - CapEx) / Σ(n_income_attr_p)
```

> 注：因季度现金流数据覆盖不全，采用年报数据替代。2 年年报等效于 8 个季度。

| 比率范围 | 含义 |
|----------|------|
| < 0 | 利润与现金流异号（极端预警） |
| 0 ~ 0.5 | 🔴 利润虚高，现金支撑严重不足 |
| 0.5 ~ 1.0 | 🟡 一般，部分利润有现金支撑 |
| 1.0 ~ 1.5 | 🟢 良好，利润有充足现金支撑 |
| > 1.5 | 🔵 保守型会计，现金流远超利润 |

### 前视偏差防护

- 回测每期仅使用公告日 ≤ 调仓日的年报
- 年报法定截止日为次年 4/30

## 四、结论与建议

### 诚信度过滤能否提升策略？


✅ **排雷有效，但选股不如FCF率**：现金实现率是一个有用的排雷指标。高诚信组（6.4%）显著跑赢低诚信组（0.7%），差异 5.7 个百分点。

但所有诚信度分组的绝对收益（最高仅 6.4%）均远低于 E版 FCF 策略（15.8%），说明：
- **诚信度作为排雷工具有效**：可以剔除利润造假/虚高标的
- **诚信度不能替代 FCF 率选股**：单纯排除低诚信公司 ≠ 选中好公司
- **建议用法**：在现有 E版 FCF 策略中，对候选池加一道诚信过滤（如要求现金实现率 > 0.3），预期可降低踩雷概率而不显著缩窄选股空间

### 注意事项

1. **净利润为负的标的被排除**：当净利润为负时，现金实现率失去意义
2. **成长股陷阱**：高成长公司资本开支大，FCF 低 ≠ 不诚信
3. **行业差异**：重资产行业天然 FCF 低，轻资产行业 FCF 高
4. **年报 = 8 季度近似**：因季度现金流数据覆盖不全，使用 2 年年报替代，精度稍降
