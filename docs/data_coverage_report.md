## 三、合理缺失说明

| 原因类型 | 数量估算 | 说明 |
|---------|:-------:|------|
| 金融/地产（FCF策略排除） | ~138只 | 银行/证券/保险/地产行业，FCF策略不使用，无需下载 |
| 股票尚未上市（历史期） | 视年份 | 某只股票2019年上市，则2015-2018年无年报，属合理缺失 |
| 退市后无年报 | 少量 | 股票退市后不再发布年报，属合理缺失 |
| Tushare数据库不收录 | 极少量 | 部分特殊股票Tushare接口查无数据，已穷举重试 |
| 季报未在期限内披露 | 极少量 | 极少数公司季报延迟披露，Tushare查无数据 |
