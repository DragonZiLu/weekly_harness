## 一、两个版本的核心差异

B版和X版共享完全相同的选样逻辑（中证800成分 + TTM口径FCF + 5年宽松OCF + PQ质量过滤 + 行业排除），
唯一差异在于选股数量和加权方式：

| 维度 | **B版（Top50精选）** | **X版（全成分）** |
|------|:---|:---|
| **选股逻辑** | FCF率排名Top50 | 全部合格公司入选（不做截断） |
| **加权方式** | FCF绝对值加权 + 10%封顶 | FCF绝对值加权 + 10%封顶 |
| **平均持仓数** | 50只 | 267只 |
| **持仓范围** | 50只 | 180 ~ 338只（随时间变化） |
| **核心理念** | FCF率最高=回报最好，精选优于分散 | FCF因子纯暴露，所有合格股都应持有 |
| **策略YAML** | `strategies/zz800_fcf/strategy.yaml` | `strategies/zz800_fcf_full_universe/strategy.yaml` |

### 选样流程（双版本共享的筛选步骤）

```
中证800全成分（800只）
  │
  ├─ 排除金融/地产（银行、证券、保险、地产开发）
  ├─ 5年OCF宽松检验（5年中≥1年OCF>0）
  ├─ PQ质量过滤（剔除非经常损益>总资产20%的尾部20%）
  ├─ TTM口径FCF = OCF(TTM) - Capex(TTM) > 0
  │
  ├── B版: FCF率排名 → 取Top50 → FCF绝对值加权(10%封顶)
  └── X版: 所有合格公司 → FCF绝对值加权(10%封顶)
```
