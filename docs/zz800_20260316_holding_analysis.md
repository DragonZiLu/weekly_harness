## 三、为什么中国海油只有4.54%FCF率却被官方选入？

### 根本原因：官方选样逻辑 ≠ 全市场FCF率排名

**我们的逻辑（B版）**：全市场非金融地产候选（~700只）→ FCF率降序 → Top50

**932368官方逻辑**（推断）：
1. 按**行业分层**选样（类似行业内FCF百分位）
2. 能源/资源行业：中国海油是石油开采行业FCF最高的标的
3. 最终组合兼顾**行业分散**，不是纯量化排名

**中国海油（600938）的FCF数据**：
```
OCF(2024年报):  2,209亿元
Capex(2024年报): 1,234亿元
FCF:              975亿元  ← 全市场绝对额最大之一
EV:             ~21,500亿元
FCF率:            4.54%   ← 因为公司体量大，FCF率看起来不高
```

**FCF率低但体量大**：中国海油FCF绝对值975亿，但EV超2万亿，所以FCF率只有4.5%。官方可能按行业内相对排名选入，而非全市场绝对FCF率。

## 四、2025Q3数据缺失问题（关键数据Bug）

**现状**：
- 2025Q3季报本地数据：仅193只（2025年10月公告）
- 2025Q2季报：935只
- 2025Q1季报：942只

**影响**：
- TTM计算公式：`TTM = 2024年报 - 2024Q3 + 2025Q3`
- 没有2025Q3 → TTM缺失 → 回退到2024年报
- 2024年报的参考期是2024Q4，而2026年3月调仓时我们已经可以用2025Q3数据

**量化影响**：
- B版Top50中大量股票用了2024年报FCF
- 如果有完整2025Q3，TTM会更新（+1季度），可能改变排名
- 特别是2024年经营情况发生变化的行业（能源、汽车、消费）

**结论**：需要补全2025Q3数据（目前193只仅覆盖10月早公告的小公司，主要大公司10月末才公告）

## 六、关键量化结论

| 指标 | 值 |
|------|---|
| Recall（同月） | **48%（24/50）** |
| 权重覆盖（24只重叠） | **60.7%** |
| MAD（权重差异） | 0.67% |
| Spearman（权重相关） | **0.973** |

**结论**：
1. **选样差异大（Recall=48%）**，但**权重相关很高（Spearman=0.973）** — 说明选中的24只股票权重分配逻辑一致
2. **官方独有26只FCF率低（平均4.48%）** vs **B版独有26只FCF率高（平均7.8%）** — 策略存在高FCF率偏向，而官方更注重行业分散
3. **2025Q3数据缺口**是潜在的改进点，补全后TTM精度提升，Recall可能改善
