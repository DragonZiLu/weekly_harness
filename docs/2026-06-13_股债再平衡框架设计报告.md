3. **"成本侵蚀复利"** — 债券 ETF 管理费仅 0.20%，远低于主动债基的 0.6-1.0%

## 二、债券基金池（三层久期阶梯）

### 2.1 选基标准

- ✅ **场内ETF** — 与现有股票持仓同平台交易，无需额外账户
- ✅ **低费率** — 管理费+托管费 ≤ 0.20%（债券ETF行业标准）
- ✅ **规模充足** — 日均成交 > 1亿，无流动性折价
- ✅ **纯债属性** — 不含权益/可转债，真正的分散工具
- ✅ **利率债** — 无信用风险，与银行股持仓零重叠

### 2.2 三层配置

| **L1** | 银华日利 | 511880 | 货币ETF | 0.08年 | ~1.45% | 0.20% | ~1200亿 | 现金管理·应急池 |

| **L3** | 富国政金债ETF | 511520 | 中长久期利率债 | 6.2年 | ~2.30% | 0.20% | ~330亿 | 降息加速器·高息差配置 |

### 2.3 为什么选这三只？

```
稳健性（利率敏感度低）  
    ↑  
    │  L1 银华日利 (久期0.08) ← 零风险，随时变现  
    │  L2 博时国开债 (久期1.3) ← 短久期，降息先涨  
    │  L3 富国政金债 (久期6.2) ← 长久期，利率下行弹性大  
    │  
    └──→ 收益潜力（利率下行弹性大）
```

**关键观点**：不使用信用债/城投债 ETF，避免与银行股（已持有大量城投贷款）重复承担同一底层风险。

## 三、股债再平衡规则

### 3.1 核心公式

```
目标股票占比 = 博格基础 
            + 息差调整 
            + 波动缓冲

其中：
  博格基础  = max(100 - 年龄, 30%)
  息差调整  = 0.10 × (当前息差 - 中性值150BP) / 100BP
  波动缓冲  = -0.05 × max(权益波动率 - 20%, 0) / 10%

最终夹逼在 [30%, 90%] 区间
```

### 3.2 规则解读

#### 博格基础（60% 基准）

40 岁 → 股票基准 60%，债券 40%。这是博格最朴素但也最稳健的建议。

#### 息差调整（动态倾斜）

| 股债息差(BP) | 市场状态 | 息差调整 | 目标股票% | 含义 |
|:----------:|---------|:------:|:--------:|------|
| >350 | 🔴 股票极度便宜 | +20% | 80% | 大举增配权益 |
| 200-350 | 🟠 股票低估 | +5~20% | 65-80% | 适度超配 |
| 100-200 | 🟡 正常区间 | -5~+5% | 55-65% | 中性格局 |
| 50-100 | 🟢 股票偏贵 | -10~-5% | 50-55% | 增配债券 |
| <50 | 🔵 股票贵 | -10% | 50% | 债券为主 |

**当前状态（2026年6月）**：中证红利股息率约 5.2%，10Y 国债 1.65%，息差 **355 BP → 股票极度便宜 → 目标股票 80.5%**。

#### 波动缓冲（风险平抑）

当市场波动加大（如沪深300年化波动率 > 20%）时，自动降低权益配置。这是博格"风险决定配置"理念的直接应用。

### 3.3 再平衡触发

| 参数 | 值 | 说明 |
|------|:--:|------|
| 检查频率 | 月度 | 每月底检查偏离度 |
| 执行频率 | 实际触发时 | 偏离 > 5% 才执行，避免频繁交易 |
| 调仓方式 | 渐进式 | 一次调整不超过总资产 20% |
| DCA 联动 | 自动 | 偏离时自动调整下期 DCA 股债分配比例 |

### 3.4 场景模拟（40岁 · 50万资产）

| 场景 | 股息率 | 息差(BP) | 目标股票 | L1现金 | L2短债 | L3长债 |
|------|:-----:|:------:|:-------:|:-----:|:-----:|:-----:|
| 熊市底部 | 7.0% | 535 | **90.0%** | 2.5% | 5.0% | 2.5% |

**关键发现**：当前 355BP 的息差意味着债券配置应该低于博格基准的 40%——只需约 20%。但这不是"不买债券"，而是等待息差收窄时再逐步回补。

## 四、与现有系统的集成

### 4.1 DCA 定投联动

```
每月定投 ¥10,000：
  基础：50% 股票 + 50% 债券
  偏离 5% → 股票/债券流量 ±10%（加速回归）

当前示例（股票低配 8.5%）：
  → 股票定投 ¥6,700 (67%)
  → 债券定投 ¥3,300 (33%)
```

当股票大幅低配时，DCA 自动增配股票流量，避免在底部卖掉债券的错误。

### 4.2 红利评分信号

股债息差同时是：
- 红利体系中 S2 维度的评分核心
- 股债再平衡的调仓信号

两者共享同一信号源，不存在冲突。

## 五、风险管理

### 5.1 利率风险

| 风险场景 | 影响 | 应对 |
|---------|------|------|
| 加息周期 | L3 长久期债下跌 | 减配 L3，增配 L1/L2 |
| 降息周期 | 债券价格上涨 | 持有 L3 获利，适时减仓转权益 |
| 信用事件 | 信用债踩踏 | **不使用信用债**，仅利率债 |

### 5.2 流动性风险

- L1 银华日利：T+0 赎回，无流动性问题
- L2/L3 日成交均 > 5亿，可承受百万级调仓

### 5.3 极端场景压力测试

| 场景 | 股票跌 | 债券表现 | 组合影响 |
|------|:-----:|---------|---------|
| 2015股灾 | -45% | +5% (L3长债) | 债券缓冲降低回撤 10% |
| 2018熊市 | -30% | +8% (降息) | 息差拉大 → 触发增配权益 |
| 2020疫情 | -15% | +3% | 快速恢复，债转股获利 |
| 债市大跌 | 不变 | -3% (L3) | 债券端回撤可控 |

## 六、实施路线图

### 第一步：建立债券底仓（即时）

```
当前资产 50 万，股票占比 72%（过于集中）

执行计划：
  卖出债券 ¥42,500 → 按 L1:L2:L3 = 25:50:25 分配
    L1 银华日利: ¥10,625  → 占比 4.9%
    L2 博时国开债: ¥21,250 → 占比 9.8%
    L3 富国政金债: ¥10,625 → 占比 4.9%
  
  目标：股票 72% → 80.5%（买入红利评分最高的标的）
```

实际上当前息差（355BP）显示股票应该 **增配而非减配**——所以正确操作是：

```
当前资产 50 万，股票占比 72% → 目标 80.5%

正确的操作方向：
  ➕ 买入权益 ¥42,500（优先 DCA 评分最高的标的）
  ➖ 卖出/减少债券 ¥42,500
```

等等——如果用户目前没有持有债券，那就直接新买入权益即可。

### 第二步：DCA 中引入债券流（月度）

每月定投 ¥10,000 中自动分配股债比例。

### 第三步：季度再平衡（自动）

每季度末运行 `rebalance_engine.py`，偏离超 5% 执行。

### 第四步：与 weekly_harness 集成（长期）

将再平衡引擎注册为 weekly_harness 的一个策略模块，与现有 DCA/红利评估共享数据。

## 八、总结

### 框架核心

```
博格基础(60%) + 息差倾斜(±20%) + 波动缓冲(-0~5%) = 目标配置(30~90%)

债券端：
  三层久期阶梯 → L1(现金) 25% + L2(短债) 50% + L3(长债) 25%

执行纪律：
  月度检查 + 5% 偏离触发 + DCA 流量自动调整
```

### 与博格思想的对应

| 博格原则 | 在本框架的落地 |
|---------|-------------|
| 成本至关重要 | 选场内 ETF，总费率 0.20%，远低于主动债基 |
| 指数化优越性 | 三只 ETF 均跟踪债券指数，不做择时选券 |
| 均值回归 | 股债息差驱动再平衡，低买高卖 |
| 债券≈年龄 | 40 岁 → 60% 基准，但息差修正不做机械执行 |
| 买入并持有 | 偏离 5% 才触发，避免频繁交易 |

> **博格**："在投资中，你得到的是你不付出的。" 
> 
> 不付出 1% 的主动管理费，你就得到了每年 1% 的额外复利。
> 不忽略债券的配置价值，你就得到了真正的分散和安全感。
