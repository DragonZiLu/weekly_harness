## 十二、核心结论

1. **万宝路真实存在但极度稀缺**：ZZ800 中仅 12 只（3%），百里挑一
2. **银行是第一产区**：6/12（50%），六大行全入 + 兴业银行
3. **银行内分化致命**：招商/宁波是红利复利（+200%+），民生/华夏是价值毁灭（-10%+），选错银行 = 天堂地狱
4. **煤炭悖论**：9/9 正收益、均值 +130%、但 0 只万宝路 — 股价涨太多稀释了股息占比
5. **持有 10 年胜率仅 64%**：不做精选就是掷骰子，36% 亏损
6. **证券是全 A 股最差赛道**：16/21 毁灭（76%），均值 -32.5%
7. **10 年期收益 99% 由股价驱动**：分红是安全垫和复利加速器，但不能替代选股
8. **"先拉数据到本地再计算"的模式验证成功**：411 只计算仅 7 秒，零 API 失败
