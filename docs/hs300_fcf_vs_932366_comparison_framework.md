# 沪深300 FCF 策略 vs 932366.CSI 官方指数 — 对比测试框架

## 1. 对比目标

量化验证我们的 FCF 选股策略与中证官方 932366.CSI（沪深300自由现金流指数）的偏差，
定位差异来源，评估策略复现质量。

## 3. 对比维度

### 3.1 成分股重合度（Recall / Precision）

```
指标:
  - Recall    = |我们 ∩ 官方| / |官方|        （我们覆盖了多少官方成分股）
  - Precision = |我们 ∩ 官方| / |我们|        （我们选了多少正确标的）
  - Jaccard   = |我们 ∩ 官方| / |我们 ∪ 官方|

逐期计算，报告: 平均值、最小值、趋势图
```

### 3.2 权重对比

```
指标:
  - Spearman 秩相关系数           （权重排序一致性）
  - Pearson 相关系数               （权重数值线性相关性）
  - 平均绝对偏差 (MAD)            （|w_ours - w_official| 的均值）
  - 最大偏差                       （单只股票最大权重差异）
  - 权重偏差分布                   （直方图 + 分位数）

逐期计算，重点关注:
  - Top 10 权重股是否一致
  - 10% 封顶规则 vs 官方无封顶 的偏差
```

### 3.3 收益对比

```
指标:
  - 年化收益率差异
  - 累计收益曲线对比图
  - 逐期收益率散点图 + 相关系数
  - 超额收益（我们的NAV - 官方TR指数）
  - 跟踪误差 (Tracking Error)
  - 信息比率 (IR = 超额收益 / 跟踪误差)

期间: 2016-06-13 ~ 2026-03-16
```

### 3.4 归因分析（差异定位）

```
逐项拆解差异来源:
  A. 行业过滤差异    → 我们剔除金融地产，官方是否也剔除？
  B. EV口径差异      → circ_mv vs total_mv vs 官方口径
  C. TTM口径差异     → 年报 vs TTM vs 官方
  D. 5yr OCF差异     → 宽松 vs 严格 vs 官方
  E. 加权方式差异    → FCF加权10%封顶 vs 官方加权方式
  F. 盈利质量过滤    → 我们有PQ筛选，官方是否有？
  G. 持仓数量差异    → 我们固定50只，官方是否也50只？
  
对每项差异:
  - 量化影响（开/关该项对重合度和收益的影响）
  - 标注: 可修复 / 设计差异 / 数据限制
```

### 3.5 标的级别诊断

```
对每个调仓期，列出:
  - 我们独有（官方没有）Top 5 权重 → 原因分析
  - 官方独有（我们没有）Top 5 权重 → 排除原因(OCF? EV? PQ? 排名不够?)
  - 权重偏差最大 Top 5 → 偏差原因(封顶? FCF率计算?)
```

## 4. 三版对比（A/B/C vs 官方）

| 版本 | EV | TTM | OCF | 预期与官方重合度 |
|------|-----|-----|-----|----------------|
| A: 原始宽松 | circ_mv | 无 | 宽松 | 低（EV口径不同） |
| B: fixed+宽松 | total_mv | 有 | 宽松 | **最高**（规则最接近） |
| C: fixed+严格 | total_mv | 有 | 严格 | 中（OCF过严排除部分） |

```
我们调仓期       →   官方权重日期（取最近可用）
2024-12-16      →   20241231
2025-03-17      →   20250331
2025-06-16      →   20250530
```
