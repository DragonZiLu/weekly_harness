## 十、结论

**D版(缓冲区±20%) vs B版(基线):**

1. ✅ **换手率大幅降低**: 43.1% → 31.3% (-27%), 首次达标<40%
2. ⚠️ **收益略降**: 年化14.58% → 14.04% (-0.54pp)
3. ✅ **风险指标相近**: 回撤/波动率/下行风险差异均<0.1pp
4. ✅ **夏普/卡玛均达标**: 夏普>0.5(0.77), 卡玛>0.5(0.80)
5. ✅ **超额vs 932366**: +2.64pp > 2pp达标
6. ✅ **与932366权重相关性略优**: Spearman 0.9963 vs 0.9957(2024-12期)
7. ❌ **信息比率不达标**: IR=-0.11(vs A版基准), 因D版和B版偏离A版方向一致

**实盘意义**: 换手率降低27%意味着交易成本节省约0.3-0.5%/年，可能弥补0.54pp的收益损失。D版在实盘场景下综合表现可能优于B版。

**改进方向**:
1. 缩小缓冲区(±15%/±10%)可进一步平衡换手率vs收益
2. 缓冲区+PQ筛选优化组合测试
3. 本地缓存复权价避免API限流
4. 600941(中国移动)等大市值标的: 可考虑对EV>2000亿的标的放宽FCF率门槛
