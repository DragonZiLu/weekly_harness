# 中证800自由现金流策略 (ZZ800 FCF)

> 对标中证800自由现金流指数 (932368.CSI)，经验证 Recall=84%，权重Spearman=0.9973

## 策略概述

从中证800成分股(沪深300+中证500)中选取 **50 只** 自由现金流率最高的非金融地产股，FCF加权（10%封顶），季度调仓。

## 编制规则

| 项目 | 内容 |
|------|------|
| 样本空间 | 中证800指数成分股 (000906.SH) = 沪深300 + 中证500 |
| 行业剔除 | 中证一级行业: 金融、房地产 (实现: 申万分类+关键词兜底) |
| 选股数量 | 50只 |
| FCF定义 | OCF(TTM) − 购建固定资产、**无形资产**和其他长期资产支付的现金(TTM) |
| EV定义 | 总市值 + 总负债 − 货币资金 |
| 5年OCF | 连续5年经营活动现金流净额 > 0 (TTM口径优先, 年报回退) |
| 盈利质量 | PQ = (OCF − 营业利润) / 总资产, 全样本空间前80% |
| 加权方式 | FCF加权, 单只上限10% (迭代封顶重分配) |
| 调仓频率 | 3/6/9/12月第二个星期五下一交易日 |
| TTM参考期 | 3月→上年Q3, 6月→当年Q1, 9月→当年Q2, 12月→当年Q3 |

### 与HS300 FCF(932366)的关键差异

| 维度 | HS300 FCF (932366) | ZZ800 FCF (932368) |
|------|-------------------|-------------------|
| 样本空间 | 沪深300(300只) | 中证800(800只) |
| 成分来源 | 全HS300 | ~19只HS300 + ~31只ZZ500 |
| **CAPEX口径** | 购建固定资产支付现金 | 购建固定资产、**无形资产**和其他长期资产支付现金 |

> ⚠️ CAPEX口径差异: 932368含"无形资产和其他长期资产", CAPEX更大 → FCF更小
> tushare字段 `c_pay_acq_const_fiolta` 已包含全部, 两个策略实际使用同一字段
> 真正差异在于: 932366官方仅用"购建固定资产"部分, tushare无法拆分

### 成分股来源分布

932368的50只成分股中，约19只来自沪深300，31只来自中证500。中证500贡献62%成分股，说明中证500中FCF率高的标的更多。

## 验证结果 (2024-12-16 调仓)

| 指标 | 值 |
|------|-----|
| Recall | **84%** (42/50) |
| 权重Spearman(封顶后) | **0.9973** |
| 缺失标的 | 8只 |
| 封顶标的 | 中国石油 21.63%→10%, 中国海油 14.67%→10%, 美的集团 8.96%→10%, 中国神华 8.99%→10% |

### 8只缺失标的根因

| 根因 | 数量 | 标的 | 可修复性 |
|------|------|------|---------|
| 行业分类(申万vs中证) | 1 | 国网英大(多元金融) | 需中证分类映射 |
| 5年OCF口径差异 | 2 | 老凤祥(2019OCF=-12.5亿), 九号公司(2021OCF=-1.6亿) | 补全Q3数据可修复 |
| PQ不达标 | 1 | 金钼股份(PQ=-0.0358) | cutoff差异 |
| 排名50以外 | 4 | 长城汽车#52, 晋控煤业#52, 永泰能源#54, 辽港股份#67 | 数据口径差异 |

## 使用方法

### 1. 命令行运行

```bash
# 生成最新一期选股
python run_fcf_strategy.py --strategy zz800_fcf

# 指定调仓日期
python run_fcf_strategy.py --strategy zz800_fcf --date 2025-06-16

# 下载缺失数据后运行
python run_fcf_strategy.py --strategy zz800_fcf --download
```

```python
from weekly_harness.fcf_universe import FcfUniverse

```

## 数据准备

中证800 = 沪深300 + 中证500, 需要两组财务数据:

| 数据类型 | HS300 | ZZ500 | 合计 |
|---------|-------|-------|------|
| 年报(cashflow/income/balance) | 300只 | 500只 | 800只 |
| 季度Q3(2023+2024) | 300只 | 500只 | 800只 |
| 指数权重 | 000300.SH | 000905.SH | 000906.SH |

数据文件位于 `data/fcf_financials/` 和 `data/index_weights/`。

## 风险提示

1. CAPEX口径差异: tushare `c_pay_acq_const_fiolta` 含无形资产, 932366仅需固定资产部分(无法拆分)
2. 申万行业分类 ≠ 中证行业分类, 可能误剔/漏剔金融地产 (国网英大为例)
3. 中证500季度数据覆盖不如HS300完整, 可能遗漏TTM口径5年OCF通过标的
4. Recall 84% 低于 HS300 FCF 的 92%, 因样本空间更大+CAPEX口径差异
