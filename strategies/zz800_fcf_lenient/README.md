# 中证800自由现金流策略 — B版 (fixed+宽松OCF)

> 版本: **B — fixed+宽松OCF**
> 对比维度: 与A版(原始宽松)相比，改用 total_mv+TTM；与C版(fixed+严格)相比，采用宽松5yr OCF规则

## 版本说明

| 版本 | EV | TTM | 5yr OCF | 年化 |
|------|-----|-----|---------|------|
| **A: 原始宽松** | circ_mv | 无 | 宽松+截断 | 14.02% |
| **B: fixed+宽松（本版本）** | total_mv | 有 | 宽松+截断 | TBD |

**本版本 (B) 的作用**：隔离 EV+TTM 变量，纯粹验证"用 total_mv 代替 circ_mv 以及引入 TTM 口径"对选股结果的影响。

## 策略规则

### 与 A 版(原始宽松)的差异

| 规则 | A版(原始宽松) | **B版(fixed+宽松)** |
|------|-------------|-------------------|
| EV计算 | `EV = circ_mv + 总负债 − 现金` | `EV = total_mv + 总负债 − 现金` ✅ |
| FCF口径 | 年报 | **TTM（最近4季累计）** ✅ |
| 5yr OCF | 宽松（缺失跳过+截断到上市年） | **宽松（同A版，不变）** |

### 与 C 版(fixed+严格)的差异

| 规则 | **B版(fixed+宽松)** | C版(fixed+严格) |
|------|-------------------|----------------|
| EV计算 | total_mv ✅ | total_mv ✅（相同） |
| FCF口径 | TTM ✅ | TTM ✅（相同） |
| 5yr OCF | **宽松（缺失跳过+截断）** | 严格（缺失=不通过） |

### 完整规则

| 项目 | 内容 |
|------|------|
| 样本空间 | 中证800指数成分股 (000906.SH) = 沪深300 + 中证500 |
| 行业剔除 | 金融、房地产（申万分类+关键词兜底） |
| 选股数量 | 50只 |
| FCF定义 | OCF(TTM) − CAPEX(TTM)，TTM = 最近4个季度累计 |
| EV定义 | **总市值(total_mv)** + 总负债 − 货币资金 |
| 5yr OCF | 宽松模式：截断到上市年，缺失年份跳过；只要已有年份OCF均>0即通过 |
| 盈利质量 | PQ = (OCF − 营业利润) / 总资产，全样本前80% |
| 加权方式 | FCF加权，单只上限10%（迭代封顶重分配） |
| 调仓频率 | 3/6/9/12月第二个星期五下一交易日 |
| TTM参考期 | 3月→上年Q3, 6月→当年Q1, 9月→当年Q2, 12月→当年Q3 |

### 5yr OCF 宽松模式说明

```python
# 宽松模式（strict_ocf=False）逻辑：
ocf_start = rep_year - 4
# 截断到上市年（允许上市不足5年的标的入选）
ocf_start = max(ocf_start, list_year)

# 检查每年OCF：
for year in range(ocf_start, rep_year + 1):
    if ocf is None:
        continue   # 缺失年份跳过（不视为不通过）
    if ocf <= 0:
        return False
return True
```

**与严格模式的关键区别**：
- 中国海油（2022年A股上市）：宽松模式截断到2022年，只检查2022-2023两年 → **可能通过**
- 严格模式：检查2019-2023年，2019-2021无A股数据 → **直接排除**
