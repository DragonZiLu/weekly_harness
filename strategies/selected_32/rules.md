# 买卖规则

> 版本 v1.0 | 2026-06-02

## 一、买入规则

### 1.1 评分驱动的目标仓位

每季度末运行 DividendCycleEvaluator，对32只标的评分（0-100分）：

```
评分 ≥ 80 → 🔥 大胆攒股 → 15% 仓位（低吸区额外+20%）
评分 65-79 → ✅ 积极布局 → 10% 仓位
评分 50-64 → 👀 观察等待 → 5% 仓位
评分 < 50 → ⏸️/🚫 暂缓/回避 → 0%（清仓）
```

### 1.2 买入触发条件

| 场景 | 条件 |
|:---|:---|
| 新买入 | 评分 ≥ 50 且当前无持仓 |
| 加仓 | 当前权重 < 目标权重 - 2%（调仓阈值） |
| 网格加仓 | 股价跌入"低吸区"（股息率 ≥ 行业买入线），权重×1.2 |

### 1.3 买入执行规则

- 每次买入至少 1 手（100股），不够 1 手不买
- 现金不足时自动卖出银华日利(511880)补充资金
- 剩余闲置资金自动扫入银华日利

## 二、卖出规则

### 2.1 卖出触发条件

| 场景 | 条件 |
|:---|:---|
| 清仓 | 评分 < 50（跌出观察线） |
| 减仓 | 当前权重 > 目标权重 + 2%（调仓阈值） |
| 网格减仓 | 股息率 < 行业减仓线，权重×0.5 |
| 风控减仓 | 单标的 > 20%（硬上限） |

### 2.2 卖出顺序

1. 先执行所有卖出（释放现金）
2. 再执行买入（使用已释放现金）
3. 每笔卖出按 100 股整数手

## 三、行业对冲规则

持有行业 A 时，配对行业 B 至少配 A 权重 × 0.5：

- **煤炭 ⇄ 火电**：煤价涨→煤炭受益，煤价跌→火电受益
- **银行 ⇄ 保险**：金融板块内均衡
- **石油 ⇄ 化工**：上下游对冲

对冲仅在配对行业有合格标的（评分 ≥ 50）时激活。
