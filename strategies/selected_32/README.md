# 精选32红利周期轮动策略

> **版本**: v1.0 (2026-06-02)  
> **状态**: 活跃  
> **策略类型**: 红利周期轮动 + 季度调仓

## 一、策略概要

**核心思想**：选择 A 股市场 32 只高股息优质标的，基于「红利周期三层估值体系」每季度评分，按评分自动分配仓位，实现红利复投的长期复利增长。

**收益目标**：年化 8-12%，最大回撤控制在 -15% 以内。

## 二、策略逻辑

### 2.1 五维评分体系（100分制）

| 维度 | 权重 | 评估内容 |
|:---|:---|:---|
| S1 股息率 | 30分 | 当期现金股息率 vs 行业历史锚 |
| S2 股债息差 | 25分 | 股息率 - 10年国债收益率(BP) |
| S3 等效分红率 | 20分 | 现金分红 + 回购收益 |
| S4 确定性/护城河 | 15分 | 行业地位、竞争壁垒 |
| S5 成长性 | 10分 | ROE + 净利润增速 |

### 2.2 评分 → 仓位映射

| 评分区间 | 信号 | 目标权重 |
|:---|---:|:---|
| ≥ 80 | 🔥 大胆攒股 | 15% (低吸区×1.2) |
| 65-79 | ✅ 积极布局 | 10% |
| 50-64 | 👀 观察等待 | 5% |
| < 50 | ⏸️/🚫 暂缓/回避 | 0% |

### 2.3 风控约束

| 约束项 | 限制 |
|:---|:---|
| 单标的上限 | ≤20% |
| 最大持仓数 | ≤15只 |
| 弱周期红利上限 | ≤40% |
| 消费成长红利上限 | ≤30% |
| 周期资源红利上限 | ≤20% |
| ETF红利上限 | ≤15% |
| 最低现金保留 | ≥5% |
| 调仓阈值 | 权重偏离 >2% 触发 |

### 2.4 行业对冲

| 行业A | 行业B | 对冲逻辑 |
|:---|:---|:---|
| 煤炭 | 火电 | 煤价涨→煤炭受益，煤价跌→火电受益 |
| 银行 | 保险 | 金融板块内均衡 |
| 石油 | 化工 | 上下游对冲 |

### 2.5 调仓频率

- **季度调仓**：每季度末最后一个周五
- 闲置现金自动买入银华日利(511880)赚取货币收益

## 三、实施方式

```bash
# 完整回测
python run_backtest.py --universe default --start 2015-01-01

# 仅生成当季调仓计划
python run_backtest.py --plan-only

# 自定义参数
python run_backtest.py --universe default \
    --max-weight 12 --mid-weight 8 --min-weight 3
```

**核心文件**：
- `dividend_evaluator.py` — 精选32标的定义 + 实时评估引擎
- `weekly_harness/strategy.py` — 策略引擎（评分→仓位→调仓指令）
- `weekly_harness/backtest.py` — 回测引擎
- `run_backtest.py` — 入口脚本
