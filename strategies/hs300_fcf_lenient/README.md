# 沪深300自由现金流策略 — B版 (fixed+宽松OCF)

> 版本: **B — fixed+宽松OCF**
> 对比维度: 与A版(原始宽松)相比，改用 total_mv+TTM；与C版(fixed+严格)相比，采用宽松5yr OCF规则

## 版本说明

| 版本 | EV | TTM | 5yr OCF | 年化 |
|------|-----|-----|---------|------|
| **A: 原始宽松** | circ_mv | 无 | 宽松+截断 | 14.69% |
| **B: fixed+宽松（本版本）** | total_mv | 有 | 宽松+截断 | 14.58% |
| C: fixed+严格 | total_mv | 有 | 严格+不截断 | 10.84% |

## 策略规则

### 与 A 版(原始宽松)的差异

| 规则 | A版(原始宽松) | **B版(fixed+宽松)** |
|------|-------------|-------------------|
| EV计算 | `EV = circ_mv + 总负债 − 现金` | `EV = total_mv + 总负债 − 现金` ✅ |
| FCF口径 | 年报 | **TTM** ✅ |
| 5yr OCF | 宽松（缺失跳过+截断到上市年） | **宽松（同A版，不变）** |

### 5yr OCF 宽松模式说明

```python
# 宽松模式（strict_ocf=False）：
ocf_start = max(rep_year - 4, list_year)  # 截断到上市年
for year in range(ocf_start, rep_year + 1):
    if ocf is None:
        continue   # 缺失年份跳过
    if ocf <= 0:
        return False
return True
```

### 关键发现

1. **B版年化14.58%，几乎与A版(14.69%)持平**：EV+TTM变化影响仅-0.11pp，说明 circ_mv→total_mv 和年报→TTM 对整体收益影响极小
2. **B版超额(vs 932366): +4.15pp**：虽然规则最接近官方932366，但仍有4.15pp超额，可能源于加权方式差异（fcf_weighted vs 指数加权）和行业过滤
3. **C版收益(10.84%)比B版低3.75pp**：严格OCF显著降低收益，排除了中国海油(10%)、中石油(10%)等大市值高FCF标的
4. **权重排序高度一致**：A→B版Spearman=0.9990，说明EV+TTM变化虽然改变了成分股(69%重合)，但权重排序几乎不变
